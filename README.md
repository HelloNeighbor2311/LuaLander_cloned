# LuaLander_cloned

A cloned Lua Lander game developed using **Unity** for learning and practicing purpose.

---

## Description

This project is made by watching tutorial on youtube and recreate it with **Unity**. I have learned some fundamental concept from it such as physics-based movement, collision detection, simple UI systems, animation systems and game management.

This is a part of my learning journey as a Game dev.

--- 

## Gameplay

- Press/hold your left mouse button to lift the spaceship upward
- Try to land on the landing pad with right speed and angle
- Score increase when successfully picking up the coin
- Recharge energy by picking up fuel
- Avoid hitting the terrain or landing with too fast speed.
- The game ends when you landing successfully.

## Feature

- Physic based spaceship movement
- Score tracking system
- Level system
- Game over and restart logic
- Simple UI(main menu, score, restart)

 ## Technologies used

 
- **Game Engine:** Unity  
- **Programming Language:** C#  
- **Version Control:** Git & GitHub
